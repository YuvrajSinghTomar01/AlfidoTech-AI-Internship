# 📌 Overview

Breast cancer is one of the most common forms of cancer worldwide. Early diagnosis plays a crucial role in improving patient survival rates and treatment outcomes.

This project applies **Machine Learning** techniques to classify breast tumors as **Benign** or **Malignant** using the **Wisconsin Breast Cancer Dataset**.

The project demonstrates a complete end-to-end Machine Learning workflow, from data preprocessing and visualization to model training, evaluation, and deployment preparation.

# 🤖 Machine Learning Models

### Logistic Regression

A linear supervised learning algorithm commonly used for binary classification problems.

### Random Forest Classifier

An ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.

### Hyperparameter Tuning

Random Forest parameters were optimized using **GridSearchCV** to achieve better model performance.
